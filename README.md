#  Video Search Engine

This project is an AI-powered video search engine that allows users to search video content using natural language queries.

---

##  Features

- Extract frames from video
- Generate embeddings using CLIP
- Store embeddings using FAISS
- Search video using text queries
- Display relevant frames with timestamps

---

##  Tech Stack

- Python
- OpenCV
- PyTorch
- Transformers (CLIP)
- FAISS
- Streamlit
